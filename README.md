# OpenApSharp

A C# / .NET translation of select parts of the python library [OpenAP](https://github.com/TUDelft-CNS-ATM/openap).

The original python library is needed (for data files) as a submodule in the project root.

Last tested with .NET 9.0.

## Building and Running

```sh
dotnet restore
dotnet build
dotnet test
```
